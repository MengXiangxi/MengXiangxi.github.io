使用北医WebVPN的简便方法

第一步，打开北医 WebVPN，尝试用北医的帐号登录。注意，是北医帐号不是北大帐号。到这里，其实就可以通过下面的各个导航项去搜索文献了，但是这还不够方便。

第二步，设置书签。很多浏览器，如 Chrome（其他 Chromium 内核的应该也可以）、FireFox、Opera 等均支持 javascript 书签功能，可以简化检索流程。在浏览器的书签栏中添加一个书签，名称随意（比如 BJMU VPN），“网址”填写下面这串代码： 

javascript:location.href = location.origin.replace(/\./g, '-') + ".webvpn.bjmu.edu.cn" + location.pathname + location.search
建议保存在书签栏。

第三步，如果有某篇文章需要北医权限，则直接按此书签按钮，网站即会直接跳转到类似 

https://xxx-xxx-xxx-nnn.webvpn.bjmu.edu.cn/xxx/xxx
这样格式的页面。这个页面就是北医 WebVPN “代理”的页面，如果北医购买了该数据库，则大概率可以在此处下载全文。

注意：

  1. 在没有登录的情况下，可能会直接转到登录页面。这样的话，在登录页面登录后，后退到文章页面再按一次书签按钮即可。
  2. 北医 Web VPN 能够支持的期刊有限，不是所有的期刊都能采用这种办法找到全文。很多情况下还得配合北大 VPN 客户端使用。个人经验是约 80% 的文献可以通过次方法找到全文。
  3. 其它一些服务，如著名的 Sci-hub，也可以用这种方式便捷地访问。
  4. 如果添加书签失败，请尝试如下 Youtube 视频或 Bilibili 中的方法添加书签。此处下载视频文件。
