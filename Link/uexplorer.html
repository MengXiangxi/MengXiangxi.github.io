Publications Related to UIH uEXPLORER PET Scanner

This is a webpage collecting publications related to a specific product, i.e. United Imaging Healthcare uEXPLORER total body PET scanner. These publications are peer-reviewed journal articles and conference abstracts which involved the uEXPLORER and its product line, or those directly discussing such equipment. For conference abstracts and proceedings, another webpage was edited to keep a record.

I am not an official source, so the list only reflects my personal opinion. I would appreciate it if anyone is willing to participate and contribute to this list.

Contribution

The list above is automatically generated from a manually-maintained BibTeX file. I rely on notifications from Google Scholar, Web of Science, and Scopus. Currently, I am keeping the BibTeX file private, but if anyone is interested, I might put it on a Git repository to receive pull requests.

For now, please contact me if you think there is a new (or missed) article involving uEXPLORER, and I will add it here after verification.

BibTeX template

The template for the BibTeX entry is like this:

@article{BIBKEY,
title={Title},
author={Author list as appears in publication},
journal={Journal},
volume={},
number={},
pages={},
year={},
doi={10.xxxx/xxxx/xxxx},
url={especially if no DOI},
pmid={if applicable},
institution={Insitution1; Insitution2},
category={#Category1; #Category2},
keywords={Tag1; Tag2}
}
There are some conventions to observe:
  • The BIBKEY is recommended to be AUTHOR+YEAR+KEYWORD, like leung2021quantitative.
  • The title shall be in sentence case. The superscript and subscript shall follow the original publication and marked by <sup></sup> or <sub></sub>.
  • The author names shall be consistent, like Last, First M, where M stands for the middle initial. Different authors are connected with and. The full author list shall be recorded, which means "et al" or "others" are not acceptable. Hint: a lot of manual adjustment may be required in this field.
  • Journal name is usually in title case and fully spelt. But there might be exceptions, such as Nuclear Instruments and Methods in Physics Research, A.
  • If there is a DOI, supply this field. If not, at least supply the URL.
  • If there is a PMID, supply this field. If not, remove it.
  • Only the leading institution and/or the institution with the uEXPLORER scanner shall be included. Different institutions are separated with a semicolon.
  • One or more (separated by a semicolon) categories can be defined for each entry. The category starts with a hashtag "#".
  • If necessary, tags can be added in the keywordsfield. Tags are separated by a semicolon.

Checklist

  • BIBKEY: correct format, uniqueness.
  • Title: sentence case, superscript and subscript, special characters.
  • Authors: completeness (Google Scholar always add "and others"), special characters.
  • Journal: case, uniqueness.
  • Year: If early-access, use "Just Accepted".
  • Pages: if multiple pages, use double dash "--".
  • DOI: not doi link (https://doi.org/).
  • PMID: check the numbering.
  • URL: link to related resources, if no DOI, provide the online article link.
  • Institution: uniqueness. Institution name shall be succinct.
  • Category: currently categories include #Algorithm, #Hardware, #Clinical, #Physiological, and #Others. If more than one category, use semicolon to separate.
  • Keywords: check for uniqueness, lower case.
    • Involving novel tracers: add tracer name
    • Check for editorial materials (editorial), reviews (review), and conference abstract (conference name)
    • Check for the following: dynamic imaging, dosimetry, deep learning, reconstruction, low dose

Known problems

  • The search function adopts a partial match fashion, resulting in the augmented search results. See this issue for an example. To avoid ambiguity, the journal name of Journal of Nuclear Medicine is thus abbreviated as J. Nucl. Med.
